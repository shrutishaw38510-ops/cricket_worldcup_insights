# Intern-Technical-Assessment
This dashboard provides a comprehensive analysis of historical Cricket World Cup data. It is designed to identify key performance patterns across countries and players using data extracted from sports archives.

# Dashboard Features
The Looker Studio report includes the following key analytical sections: -> Country Performance: A visual breakdown of total runs scored by major cricketing nations, including India, Australia, New Zealand, South Africa, and England. -> Detailed Player Analytics: A comprehensive data table tracking individual player performance metrics such as Total Runs, Wide Balls, and No Balls. -> Highlights: Rohit Sharma (271 runs), Martin Guptill (237 runs), and Virat Kohli (199 runs). -> Player vs. Player Comparison: Side-by-side bar charts to compare scoring outputs between specific players. -> Historical Trends: A line chart visualizing the highest runs in single innings over a timeline including the 2011, 2015, 2019, and 2023 World Cups. -> Geospatial Insights (Venues): A world map visualization identifying the best performing venues and countries involved in the tournament.

# Tools & Technologies
-> Visualization Tool: Google Looker Studio. -> Data Source: Historical data inspired by Wikipedia and ESPNCricInfo archives. -> Metrics Tracked: Runs, Wide Balls, No Balls, and Venue Locations.
